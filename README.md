### **FaceSwap Studio** is a modern, browser-based AR experience built with **Snap Camera Kit**, showcasing high-quality real-time face swapping, template-based video swaps, and user-uploaded media processing, all without installing an app.

- Live Face Swap Screen: This screen features the primary real-time face swapping experience, showing live camera feeds, a visual debug view for tracking, and a button to switch cameras.
- Template Selection Screen: This screen displays a library of video templates with previews, allowing users to choose one for an automatic face swap.
- Upload Media Screen: This screen provides functionality for users to upload their photos or short videos for face swapping, with clear export options for different media types.
